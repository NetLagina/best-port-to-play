    Final Fantasy III #

    Platforms #

    Famicom, Nintendo DS, Wii Virtual Console, iOS, Android, PSP, Ouya, Windows Phone, Wii U Virtual Console, Nintendo 3DS Virtual Console, Windows, NES Classic Edition, Android, IOS

    Fullest version #

    none or any

    Comment #

    There are 3 various versions of the game: Original (Famicom, all Virtual Consoles, NES Classic Edition), 3D Remake (Nintendo DS, iOS (2011), Android (2012), PSP, Ouya, Windows Phone, Windows (2014)) and 2D (pixel) Remastered (iOS (2021), Android (2021), Windows (2021)). All of them have various gameplay features, graphic style, etc.

    Most stable version #

    Nintendo DS, iOS, Android, PSP, Ouya, Windows Phone, Windows, NES Classic Edition, Android, IOS

    Best look #

    3D Remake: iOS (2011), Android (2012), Ouya, Windows Phone, Windows (2014). 2D Remaster: iOS (2021), Android (2021), Windows (2021)

    Comment #

    3D Remake version – full 3D remake. 2D remaster version looks as polished classic 2D

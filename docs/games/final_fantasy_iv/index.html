    Final Fantasy IV #

    Also known as #

    Final Fantasy IV Easy Type, Final Fantasy Collection, Final Fantasy IV Advance, Final Fantasy IV: The Complete Collection, Final Fantasy 25th Anniversary Ultimate Box, “This is not a Final Fantasy II?”

    Platforms #

    SNES, Super Famicom, PS, WonderSwan Color, GBA, Nintendo DS, Wii Virtual Console, PSP, iOS, Android, Windows, PSOne Classic, PS3, PS Vita, Wii U Virtual Console

    Fullest version #

    None. Closest to the most complete version: PSP (The Complete Collection), iOS (3D), Android (3D), Windows (3D)

    Comment #

    There are 3 groups of versions: original (SNES, Super Famicom, PS, WonderSwan Color, GBA, Wii, PSP (The Complete Collection), PSOne Classic, PS3, PS Vita, PSP, Wii U), 3D-Remake (Nintendo DS, iOS, Android, Windows) and 2D-Remaster (iOS, Android, Windows). Only PSP version has prequel “Interlude”. Only iOS/Android/Windows 3D-versions have “The After Years”

    Most stable version #

    PS, PS3, PS Vita, PSP, Nintendo DS, iOS, Android, Windows

    Comment #

    SNES, Super Famicom, GBA (GBA only JP v1.0 and NA) versions have many bugs and translation problems

    Best look #

    PSP (Final Fantasy IV: The Complete Collection), iOS, Android, Windows (both 3D and 2D-Remaster)

    Comment #

    3D Remake version – full 3D remake. 2D remaster version looks as polished classic 2D
